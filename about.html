About Crypto Price Tracker

The Crypto Price Tracker is a modern web application designed to provide real-time information about cryptocurrencies such as Bitcoin, Ethereum, and Binance Coin.

This platform helps users monitor market trends, price changes, and important financial data in a simple and interactive way.

🚀 Features

  • 📊 Real-time cryptocurrency prices
  • 📈 Interactive percentage graphs (1H, 24H, 7D)
  • 🔍 Search functionality
  • 🔄 Auto-refresh every 60 seconds
  • 💰 Market data (volume, market cap, supply)

🛠️ Technologies Used

  • HTML, CSS, JavaScript
  • CoinGecko API
  • Chart.js for graphs

🎯 Purpose

This project is built for learning web development and API integration. It also helps users quickly understand cryptocurrency market behavior.

⚠️ Disclaimer

This project is for educational purposes only. Cryptocurrency markets are volatile. Always do your own research before investing.
